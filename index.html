Inlämningsuppgift 2

Header image skalas med bredden av viewport. Rubriken ovanpå bilden skalas med clamp. De flesta CSS-klasser och subklasser är namngivna enligt BEM. Navigationsmenyn använder a:hover för att fälla ut undermenyer om skärmen är bredare än 600 pixlar. På mindre skärmar används en dold checkbox-tag för att fälla ut och in undermenyer.

Detta är så klart inte perfekt då många touch-enheter har betydligt högre upplösning än så. Men utan javascript verkar det vara omöjligt att säkert identifiera om klienten stödjer hover eller touch. Media quaries för :hover stöds av de flesta webbläsare men inte av Firefox.

Sidans layout i stort är gjord med grid och layouten för Portfolio-sektionen är gjord med flex box. Jag har inte tagit med något från Bootstrap-ramverket. Sidan responsiv på så vis att den ändrar layout, skalar om vissa element och ändrar meny när bredden understiger 600px. HTML och CSS validerar enligt W3C:s valideringstjänster.
